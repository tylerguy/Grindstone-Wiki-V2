From these Numbers, you would put these throught a polybius square using the arrangement on the right to get this:

KFCOOWECPOKFMFQGMQNVCRVQLCTZFSZYMAHPMOAWCVZ

From there, you simply put this through a Viginere substitution cipher using the keyword 'grindstone' to get:

EOUBLELOCK EOES NOT CARW AND IK ALSO THE NEXT ANSWER

After some cleanup, this translates to: Doublelock does not care and is also the next answer

Meaning the Answer for the first objective is: DOUBLELOCK
